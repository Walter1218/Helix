# 剧本写作规范

## 核心概念

### 最小出品单位：轮次（Round）

**轮次是剧本的最小出品单位，不是场景。**

- 每轮独立成片，可单独输出为一段完整视频
- 多轮可合并为一集长视频
- 场景是轮次内部的细分结构，不独立出品

### 概念层级

```
集（Episode）
  └── 轮次（Round）← 最小出品单位，独立成片
        ├── 场景 1（Scene）← 轮次内部细分
        ├── 场景 2
        └── 场景 3
```

### 轮次的独立性

每轮必须具备：

1. **明确开场** — 观众能快速进入状态
2. **完整叙事弧** — 有起承转合，不依赖其他轮次理解剧情
3. **可闭合结尾** — 可以是阶段性收束、悬念钩子或情感高潮
4. **独立情绪基调** — 每轮有主导情绪，不与其它轮次混淆

### 轮次闭合方式

| 闭合类型 | 说明 | 适用场景 |
|---------|------|---------|
| 阶段收束 | 事件告一段落，情绪落地 | 日常情节、温馨场景 |
| 悬念钩子 | 留下未解之谜，吸引看下一轮 | 反转、发现、冲突升级 |
| 情感高潮 | 在情绪最高点收束 | 感人场景、燃点场景 |
| 节奏切分 | 在节奏转换点分轮 | 长剧情、多线叙事 |

## 一、画面描述基本原则

### 1.1 空间自洽

- 同一场景内，角色位置关系必须前后一致
- 移动需有明确路径，不能瞬移
- 新场景需用文字重建空间锚点（先写场景，再写角色位置）

### 1.2 角色首次出场

- 首次出场必须包含完整外貌描述：发型、发色、服装、配饰、体型
- 后续出场可用名字引用，但场景切换后需简要重提服装（确认是否换装）
- 多人场景先写全景（谁在场、位置关系），再切近景

### 1.3 动作链完整

- 动作前因后果完整，不跳步
- 手部动作必须说明哪只手
- 连续动作用自然语言串联，避免堆砌逗号短句

## 二、分镜写法

### 2.1 格式

```
### 场景 N：场景标题

**镜头 X：镜头类型 · 角色/物体 · 动作**

文字描述。画面构成：主体在画面中的位置和状态。动作描述：具体的身体动作和表情变化。环境细节：背景元素、光线氛围。
```

### 2.2 镜头类型

| 镜头类型 | 用途 | 描述要求 |
|---------|------|---------|
| 远景 | 建立环境 | 描述整体场景氛围、光线、色调 |
| 全景 | 展示空间和角色关系 | 说明角色在空间中的位置分布 |
| 中景 | 展示角色动作 | 包含角色上半身和周围环境 |
| 近景 | 展示表情和细节 | 聚焦面部表情或手部动作 |
| 特写 | 强调关键物体或情绪 | 只包含物体或面部局部 |

### 2.3 动作连贯性

- 相邻镜头之间动作有因果关系
- 避免逻辑矛盾（如：左手拿杯子 → 下一个镜头右手拿杯子）
- 动作幅度大的镜头之间需要过渡镜头

## 三、时长控制

- 每个镜头建议 2-4 秒，避免超过 6 秒
- 动作复杂的镜头可适当延长，但需说明节奏（快/慢）
- 每轮（Round）建议 4-8 个镜头
- 每轮时长建议 1-3 分钟

## 四、轮次间衔接

### 4.1 情绪过渡

- 轮次之间情绪变化需有过渡，避免突兀跳跃
- 可通过轮次最后一个镜头的色调/节奏暗示下一轮基调

### 4.2 信息传递

- 每轮开头不假设观众记得上一轮细节
- 关键信息在轮次内自然重提（通过角色对话或动作）
- 但不要过度解释，保持叙事节奏

## 五、多轮次剧本结构

### 5.1 结构示例

```
# 第 X 集：[集标题]

## 轮次 1：[轮次标题]
> 情绪基调：xxx | 闭合方式：xxx

### 场景 1：[场景标题]
**镜头 1：...**
...

### 场景 2：[场景标题]
**镜头 1：...**
...

---

## 轮次 2：[轮次标题]
> 情绪基调：xxx | 闭合方式：xxx

### 场景 1：[场景标题]
**镜头 1：...**
...
```

### 5.2 轮次选择建议

- 按情绪段落分轮：一段完整的情绪起伏 = 一轮
- 按场景转换分轮：主要场景切换 = 新轮次
- 按叙事节奏分轮：节奏明显变化 = 新轮次
- 按闭合点分轮：有明确的阶段性收束或悬念 = 轮次结束

## 六、避免的问题

### 空间错乱
- ❌ "小猫在桌子上" → "小猫在地板上"（没有移动描述）
- ✅ "小猫在桌子上" → "小猫跳下桌子，落在地板上"

### 动作矛盾
- ❌ "左手端起咖啡" → "右手放下咖啡杯"
- ✅ "左手端起咖啡" → "左手将咖啡送到嘴边" → "左手放下咖啡杯"

### 角色描述缺失
- ❌ "小猫看了看窗外"（新场景中没有描述小猫当前状态）
- ✅ "小猫蹲在窗台上，转头望向窗外"

### 轮次不独立
- ❌ 轮次 2 开头需要观众记得轮次 1 的某个细节才能理解
- ✅ 轮次 2 通过对话或画面自然重提必要信息
