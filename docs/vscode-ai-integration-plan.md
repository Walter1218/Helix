## 技术细节

### API 端点清单

| 端点 | 方法 | 用途 | 优先级 |
|------|------|------|--------|
| `/session` | POST | 创建会话 | P0 |
| `/session/:id/message` | POST | 发送消息 | P0 |
| `/session/:id/message` | GET | 获取消息历史 | P1 |
| `/session` | GET | 获取会话列表 | P1 |
| `/session/:id/abort` | POST | 中断生成 | P2 |
| `/permission/:id/approve` | POST | 批准权限 | P2 |
| `/project/files` | GET | 文件列表 | P2 |

### 通信流程

```
用户输入消息
    ↓
helix-welcome.html (sendMessage)
    ↓
Bridge Script (拦截 fetch)
    ↓
VS Code Extension (panel.ts message handler)
    ↓
HelixServer (HTTP 转发)
    ↓
opencode CLI (处理请求)
    ↓
AI 模型 (生成响应)
    ↓
流式返回 → Extension → Bridge → UI 更新
```
